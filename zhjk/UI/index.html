课程大纲

第一阶段

了解互联网相关知识，掌握网页开发相关基础知识，掌握网页开发软件使用，理解各种HTML标签，熟练应用html书写相关网页案例。
1、html标签
2、CSS样式表
3、基本图形处理

第二阶段

能够深刻的认识PS基本概念，了解PS软件的基本功能及适用范围，对平面设计、网页设计、UI设计、网页前端的简单认识。
1、ps软件操作
2、平面构成、色彩构成、立体构成
3、平面设计实训，AI 软件操作

第三阶段

学习WEB设计规范学习页面配色及排版设计，掌握配色技巧及常用搭配，并运用到实际案例中。
1、企业网站 WEB 端
2、服装类电商 WEB 端
3、综合实训 WEB 端

第四阶段

主要讲解UI界面设计，以设计理论为基础，项目实战为核心
1、移动端设计及交互规范
2、小工具 APP
3、新闻资讯阅读类 APP

第五阶段

主要讲解UI界面设计，以设计理论为基础，项目实战为核心
1、动效设计及 AE 工具
2、服装，生鲜，农产品电商类综合实训
